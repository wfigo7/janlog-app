# Janlog リリースノート

## バージョン 0.1.0 - 初期MVPリリース

**リリース日:** 2025年11月04日

### 概要

Janlogの初期MVPバージョンをリリースしました。フリー雀荘やセット麻雀の成績を個人用に記録・集計できるモバイルアプリです。

### 主要機能

#### 対局記録
- 3人麻雀・4人麻雀の対局記録
- 3つの入力方式に対応
  - 順位+最終ポイント
  - 順位+素点（ウマ・オカ自動計算）
  - 仮ポイント（順位分布のみ）
- 対局日時、会場、メモの記録

#### 成績統計
- 対局数、平均順位、トップ率、ラス率など
- 累積ポイント、チップ合計
- 期間フィルター（全期間、今月、先月、今年、去年）
- ゲームモード別フィルター（3人麻雀、4人麻雀）

#### 成績履歴
- 過去の対局を一覧表示
- 過去の対局の編集・削除

#### ルール管理
- ルールセット（ウマ・オカ設定）の管理
- 対局登録時のルール選択
- 素点からの自動ポイント計算

#### 認証
- Cognito招待制ユーザー認証
- メール+パスワード認証

### 技術スタック

- **フロントエンド:** Expo (React Native)
- **バックエンド:** FastAPI on Lambda Web Adapter
- **データベース:** DynamoDB シングルテーブル設計
- **認証:** Amazon Cognito User Pool
- **インフラ:** AWS CDK

### 環境分離

- **local環境:** 開発者ローカル、静的JWT認証、DynamoDB Local
- **development環境:** MVP公開用、Cognito認証、AWS DynamoDB
- **production環境:** 本番用（将来実装）

### バージョン管理

このリリースから、セマンティックバージョニングに基づくバージョン管理体系を確立しました。詳細は`VERSIONING.md`を参照してください。

### 既知の制限事項

- プレリリース段階（バージョン0.x.y）のため、APIの変更が発生する可能性があります
- 個人の成績のみ管理（他人の成績は管理しません）
- 3人麻雀・4人麻雀の成績は混ぜません
